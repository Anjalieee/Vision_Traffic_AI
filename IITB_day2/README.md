# VisionTrafficAI (Step 1 - Data Collection)

This project is the initial phase of building an intelligent traffic monitoring system.

## Features
- Automated image scraping using Selenium
- Multi-label support via label.txt
- Saves images locally

## Setup

pip install -r requirements.txt

## Run

python main.py

## Notes
- Images saved in /images
- Requires Chrome browser

## Future Scope
- Vehicle detection (YOLO)
- Speed tracking
- License plate recognition
- Automated fine system
